Detailed Description

Auxiliary class for Archive manipulation.

The serialization "file" inherit from the mother class cAr2007. This class is accessible via the the cAuxAr2007 , the automitazation of calling level (usefull for example in XML pretty printing) is done by constructor and destructor of cAuxAr2007.
