Detailed Description

template<class Type>
class MMVII::cOrderedPair< Type >

Pair where we want (a,b) == (b,a)

cOrderedPair are pretty match like pair<T,T> , the main difference being that they modelise symetric graph. To assure that they are always in a single way, we force V1 <= V2
