Detailed Description

Class to register current state of memory.

cMemState memorize a summary of memory state container the number of object, the size allocacted and some majic number; it allow to check if between 2 times, all what was allocated was desallocated

Typical use cMemState aState = cMemManager::CurState(); { .. do some stuff } Assert(aState==cMemManager::CurState()); or cMemManager::CheckRestoration(aState);
