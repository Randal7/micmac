Detailed Description

Implementation of serialisation servive.

This serialization principle is very close (at least inspired by) boost one. In fact I hesitated to use boost, but could not find a way satisfying for handling optional value in XML (accept to read old xml file when the structure has grow with optionnale value).

serializing a class, consist essentially to describe serially all its data. This done by defining a function AddData that calling seriall the AddData of its member. See cTestSerial0, cTestSerial1 and cTestSerial2 at the end of this file for examples.
